# Platform Page Package Format

Every run emits the same three outputs:

1. `platform-design-system.json`
2. `shell-variant-<page-id>.json`
3. `platform-design-contract.md`

The markdown contract keeps this section order:

1. `# Platform Page Design Contract`
2. `## Scope`
3. `## Required Result`
4. `## Allowed Differences`
5. `## Requested Changes`
6. `## Shell Contract`
7. `## Theme Tokens`
8. `## Viewport Set`
9. `## Component Inventory`
10. `## Acceptance Rule`

Do not change the output type. Add extra notes outside the package if needed.
