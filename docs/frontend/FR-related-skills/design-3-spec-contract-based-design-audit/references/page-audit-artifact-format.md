# Page Audit Artifact Format

Each run emits:

1. `page-contract-audit.json`
2. `page-contract-audit.md`

The markdown report keeps this section order:

1. `# Page Contract Audit Report`
2. `## Scope`
3. `## Required Behavior`
4. `## Summary`
5. `## Source Findings`
6. `## Rendered Findings`
7. `## Fixes Applied`

The report exists to make one-page audits repeatable and reviewable.
