# Page Reproduction Contract Format

The script emits this fixed section order:

1. `# Page Reproduction Contract`
2. `## Source`
3. `## Required Result`
4. `## Requested Changes`
5. `## Fixed Layout Contract`
6. `## Section Inventory`
7. `## Live Reference Rule`
8. `## Acceptance Rule`

Do not change the output type. Add follow-up notes outside the contract if needed.
