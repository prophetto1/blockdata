Legal-10's build pipeline draws from eight primary legal data sources:

| Source | Content | Coverage |
|--------|---------|----------|
| **SCDB** | Supreme Court case metadata, votes, dispositions | 1791–present |
| **CAP** | Full-text case opinions from all U.S. jurisdictions | Historical–2020 |
| **Shepard's** | Citation treatment edges (positive/negative/distinguished) | Cross-jurisdictional |
| **Oyez** | Oral argument metadata and summaries | SCOTUS |
| **Fowler** | Authority scores for Supreme Court opinions | SCOTUS |
| **Martin-Quinn** | Ideological scores for justices | SCOTUS |
| **Songer** | Appellate court case attributes | U.S. Courts of Appeals |
| **CourtListener** | Cross-reference IDs and supplemental metadata | All jurisdictions |
