---
title: "Scripts provider"
description: "Python, Shell, and Node.js script execution with environment variables, pre-commands, and output file capture."
---

The scripts provider runs arbitrary code in isolated subprocesses. Each plugin accepts either inline script content or a list of shell commands, with support for environment variables, setup commands, and configurable timeouts.

3 plugins, 7 task types.

---

## Python

Executes Python scripts via `python3 -c`.

**Task types:**
- `io.kestra.plugin.scripts.python.Script`
- `io.kestra.plugin.scripts.python.Commands`

### Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `script` | string | no | — | Python script content. Supports `{{ expressions }}`. |
| `commands` | string[] | no | — | List of shell commands to run (alternative to `script`). Each supports `{{ expressions }}`. |
| `env` | object | no | — | Environment variables. Values support `{{ expressions }}`. |
| `beforeCommands` | string[] | no | — | Setup commands to run before the script (e.g. `pip install pandas`). |
| `timeout` | integer | no | `300` | Timeout in seconds. |
| `outputFiles` | string[] | no | — | Glob patterns for output files to capture. |

Provide either `script` or `commands`, not both. If both are given, `script` takes priority.

### Example

```json
{
  "task_type": "io.kestra.plugin.scripts.python.Script",
  "params": {
    "script": "import json\ndata = {'count': 42}\nprint(json.dumps(data))",
    "env": {
      "API_KEY": "{{ secrets.api_key }}"
    },
    "beforeCommands": ["pip install requests"],
    "timeout": 60
  }
}
```

### Response

```json
{
  "output": {
    "data": {
      "exitCode": 0,
      "stdout": "{\"count\": 42}\n",
      "stderr": ""
    },
    "state": "SUCCESS",
    "logs": ["Python script completed (18 bytes output)"]
  }
}
```

Non-zero exit codes return `FAILED` with `exitCode`, `stdout`, and `stderr` in `data`.

## Shell

Executes shell/bash scripts via `/bin/bash -c` (configurable).

**Task types:**
- `io.kestra.plugin.scripts.shell.Script`
- `io.kestra.plugin.scripts.shell.Commands`
- `io.kestra.core.tasks.scripts.Bash`

### Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `script` | string | no | — | Shell script content. Supports `{{ expressions }}`. |
| `commands` | string[] | no | — | List of commands to run (alternative to `script`). |
| `interpreter` | string[] | no | `["/bin/bash", "-c"]` | Interpreter command. Change for `sh`, `zsh`, etc. |
| `env` | object | no | — | Environment variables. |
| `timeout` | integer | no | `300` | Timeout in seconds. |

### Example

```json
{
  "task_type": "io.kestra.plugin.scripts.shell.Script",
  "params": {
    "script": "echo \"Files: $(ls /data | wc -l)\"",
    "env": {
      "DATA_DIR": "/data/incoming"
    }
  }
}
```

### Response

```json
{
  "output": {
    "data": {
      "exitCode": 0,
      "stdout": "Files: 15\n",
      "stderr": ""
    },
    "state": "SUCCESS",
    "logs": ["Shell script completed (10 bytes output)"]
  }
}
```

## Node

Executes Node.js scripts via `node -e`.

**Task types:**
- `io.kestra.plugin.scripts.node.Script`
- `io.kestra.plugin.scripts.node.Commands`

### Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `script` | string | no | — | Node.js script content. Supports `{{ expressions }}`. |
| `commands` | string[] | no | — | List of commands to run (alternative to `script`). |
| `env` | object | no | — | Environment variables. |
| `timeout` | integer | no | `300` | Timeout in seconds. |

### Example

```json
{
  "task_type": "io.kestra.plugin.scripts.node.Script",
  "params": {
    "script": "const data = { status: 'ok' }; console.log(JSON.stringify(data));",
    "beforeCommands": ["npm install cheerio"],
    "timeout": 120
  }
}
```

### Response

```json
{
  "output": {
    "data": {
      "exitCode": 0,
      "stdout": "{\"status\":\"ok\"}\n",
      "stderr": ""
    },
    "state": "SUCCESS",
    "logs": ["Node script completed (16 bytes output)"]
  }
}
```
