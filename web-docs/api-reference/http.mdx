title: "HTTP provider"
icon: "network"
description: "HTTP request and file download plugins with auth support, template rendering, and response truncation."
---

The HTTP provider makes outbound HTTP calls. Supports GET/POST/PUT/PATCH/DELETE with Basic and Bearer auth, template rendering in URLs and headers, and automatic response truncation for large payloads.

2 plugins, 4 task types.

---

## Request

General-purpose HTTP request. Returns status code, headers, and body.

**Task types:**
- `io.kestra.plugin.core.http.Request`
- `io.kestra.core.tasks.http.Request`

### Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `uri` | string | yes | — | URL to request. Supports `{{ expressions }}`. |
| `method` | string | no | `GET` | HTTP method: `GET`, `POST`, `PUT`, `PATCH`, `DELETE`, `HEAD`, `OPTIONS`. |
| `headers` | object | no | — | Request headers. Values support `{{ expressions }}`. |
| `body` | string | no | — | Request body for POST/PUT/PATCH. Strings are template-rendered; objects are sent as JSON. |
| `formData` | object | no | — | Form-encoded data (`application/x-www-form-urlencoded`). Values support `{{ expressions }}`. |
| `auth` | object | no | — | Auth config (see below). |
| `timeout` | integer | no | `30` | Request timeout in seconds. |

### Auth config

The `auth` object supports two modes:

**Basic auth:**
```json
{
  "auth": {
    "type": "BASIC",
    "username": "user",
    "password": "pass"
  }
}
```

**Bearer token:**
```json
{
  "auth": {
    "type": "BEARER",
    "token": "eyJhbGciOi..."
  }
}
```

### Example

```json
{
  "task_type": "io.kestra.plugin.core.http.Request",
  "params": {
    "uri": "https://api.example.com/documents/{{ outputs.upload.doc_id }}",
    "method": "GET",
    "headers": {
      "Accept": "application/json"
    },
    "timeout": 15
  }
}
```

### Response

```json
{
  "output": {
    "data": {
      "statusCode": 200,
      "headers": {
        "content-type": "application/json",
        "x-request-id": "abc-123"
      },
      "body": "{\"id\": 42, \"title\": \"Document\"}",
      "uri": "https://api.example.com/documents/42"
    },
    "state": "SUCCESS",
    "logs": ["HTTP GET https://api.example.com/documents/42 → 200"]
  }
}
```

State is `WARNING` for HTTP 4xx/5xx responses (not `FAILED` — the request itself succeeded).

Response bodies larger than 10 MB are truncated with `...[truncated]`.

## Download

Downloads a file. If Supabase Storage is configured, uploads the file there and returns the storage URL. Otherwise returns the source URL with content metadata.

**Task types:**
- `io.kestra.plugin.core.http.Download`
- `io.kestra.core.tasks.http.Download`

### Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `uri` | string | yes | — | URL to download. |
| `headers` | object | no | — | Request headers. Values support `{{ expressions }}`. |

Download has a fixed 120-second timeout. Non-2xx responses raise an error (returns `FAILED`).

### Example

```json
{
  "task_type": "io.kestra.plugin.core.http.Download",
  "params": {
    "uri": "https://storage.courtlistener.com/bulk/opinions-2024-01-01.csv.bz2",
    "headers": {
      "User-Agent": "blockdata-pipeline/1.0"
    }
  }
}
```

### Response (with Supabase Storage)

```json
{
  "output": {
    "data": {
      "uri": "https://dbdzzhshmigewyprahej.supabase.co/storage/v1/object/public/pipeline/downloads/exec-123/opinions-2024-01-01.csv.bz2",
      "contentType": "application/x-bzip2",
      "size": 2147483648
    },
    "state": "SUCCESS",
    "logs": ["Downloaded https://storage.courtlistener.com/bulk/opinions-2024-01-01.csv.bz2 (2147483648 bytes) → https://..."]
  }
}
```

### Response (without Supabase Storage)

```json
{
  "output": {
    "data": {
      "uri": "https://storage.courtlistener.com/bulk/opinions-2024-01-01.csv.bz2",
      "contentType": "application/x-bzip2",
      "size": 2147483648
    },
    "state": "SUCCESS",
    "logs": ["Downloaded https://storage.courtlistener.com/bulk/opinions-2024-01-01.csv.bz2 (2147483648 bytes)"]
  }
}
```
