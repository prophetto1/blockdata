This section consolidates the contracts that are currently split across implementation files and older internal notes.

It focuses on the docs app shell that is live in this repo today:

- the Starlight shell overrides wired in `astro.config.mjs`
- the runtime shell-state and file-selection contracts under `src/lib/docs`
- the active font, spacing, and color tokens defined in `src/styles/global.css`
- the editor-surface token bridge used by the split editor

## Current Drift To Keep In Mind

Two older documents remain useful, but they are not fully aligned with the live code:

- `src/content/docs/internal/shell-contract-spec.md` still describes a `Source | Keystatic` editor toggle, while the implemented toggle is `Source | Rich`.
- `docs/docs/design-system.md` describes a broader Tailwind token system from a different app surface. It is useful background, but `src/styles/global.css` is the live token source for this docs site.
